Ftrace的简单使用

跟踪 smp_apic_timer_interrupt 的函数调用过程

1、 cd /sys/kernel/debug/tracing
2、 echo function_graph > current_tracer
3、 echo smp_apic_timer_interrupt > set_graph_function
4、 cat trace
或者可以把 trace导出然后分析 cat trace > ~/trace.data
